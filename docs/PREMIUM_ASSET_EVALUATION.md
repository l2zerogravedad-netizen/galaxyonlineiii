## Critical Warnings

### ⚠️ Unity Asset Store EULA
> "The licensor grants to the Licensee a non-exclusive, worldwide license to use the Asset for commercial purposes in a video game, simulation, or interactive media product." — BUT — "The Licensee shall not redistribute the Asset or any derivative of the Asset in a way that would allow a third party to use the Asset without first purchasing it from the Unity Asset Store."

**Problema:** Si tu juego es WebGL y los assets pueden ser extraídos del build (lo cual es trivial con WebGL), hay un riesgo de "redistribución indirecta". La licencia no está diseñada para WebGL donde los assets son visibles en el cliente.

**Veredicto:** Evitar Unity Asset Store para juegos WebGL puros.

### ⚠️ Fab / UE5 Assets
> "The Licensor grants to the Licensee a non-exclusive, worldwide license to use the Assets in video games." — Standard Fab License

**Problema:** Los assets Fab están optimizados para UE5 (Lumen, Nanite, materiales `.uasset`). Para WebGL necesitas:
1. Importar a UE5 (requiere cuenta Epic)
2. Exportar a FBX desde UE5 (pérdida de materiales PBR)
3. Recrear materiales en Blender/Three.js

**Veredicto:** Usar solo si ya tienes pipeline UE5 o estás dispuesto a hacer export manual.

### ⚠️ Sketchfab Store
> "The license type is indicated on the product page. Standard licenses permit use in video games. Editorial licenses do not."

**Problema:** Cada modelo tiene licencia diferente. Algunos autores ponen "Editorial" sin saber que impide uso en juegos.

**Veredicto:** Verificar LICENCIA INDIVIDUAL de cada modelo antes de comprar.

## Blender Workflow for KitBash3D Assets

```
1. Download from KitBash3D (native .blend files)
2. Open in Blender 4.x
3. Check scale (usually 1 unit = 1 meter)
4. Apply bevels (0.02-0.05m)
5. Add panel lines with inset/extrude
6. Replace materials with project PBR library
7. Add emissive lights for sci-fi look
8. Export to GLB for WebGL
9. Place in assets/buildings/processed/
```

**No conversion needed from other formats** — KitBash3D ya incluye .blend nativo.
